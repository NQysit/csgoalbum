Changelog

v0.3.1 (beta) Released: 2015-10-25

  • Now players with '@' in their nick are correctly listed. (Thanks to @abbld)
  • Now Cluj Sticker count is reset when a new profile is loaded.

v0.3 (beta) Released: 2015-10-25

  • Now stickers are served from our host.
  • Added option to export current album to imgur
  • Added custom urls. Now you can share an album by sending its url.
  • Small interface changes.
  • Gotta catch'em all

v0.2 (beta) Released: 2015-10-21

  • Added stickers from DreamHack Cluj-Napoca 2015
  • Added Twitter button.
  • Improved application performance.

Fixed some minor bugs :

  • Iceweasel browser no longer showing 'indexOf' error.
  • Fixed some misspelled words.

v0.1 (beta) Released: 2015-10-20

First version has been released. Here are some functionalities:

  • Fetch your profile using your custom SteamID.
  • All major stickers were added.
  • Possibility of disabling animations while passing mouse over stickers.
